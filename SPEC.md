# Lway - 个人成长与习惯养成应用

## 1. 项目概述

- **项目名称**：Lway
- **项目释义**：Long Way / Life Way (人生路漫漫，唯有长期主义与自我重塑)
- **核心标语**：
  - English: Life is a long road. Walk it your way.
  - 中文：人生路漫漫，走你自己的路。
- **项目类型**：原生 Android 应用
- **核心功能**：专注个人成长与习惯养成的沉浸式体验平台，涵盖体态纠正、交易复盘、早睡早起、锻炼、阅读五大习惯领域
- **目标用户**：追求自我提升、注重自律习惯养成的个人用户

## 2. 技术规格

### 2.1 技术栈
- **语言**：Kotlin 1.9.x
- **UI 框架**：Jetpack Compose (Material 3)
- **最低 SDK**：26 (Android 8.0)
- **目标 SDK**：34 (Android 14)
- **架构**：MVVM + Clean Architecture
- **依赖注入**：Hilt
- **本地数据库**：Room
- **异步处理**：Kotlin Coroutines + Flow
- **导航**：Jetpack Navigation Compose

### 2.2 数据存储
- 本地：Room SQLite
- 云端：Firebase (Auth + Firestore)
- 文件：应用内部存储 + MediaStore

## 3. MVP 功能模块

### 3.1 习惯打卡与时长追踪
- 五大习惯领域：体态纠正、交易复盘、早睡早起、锻炼、阅读
- 记录维度：
  - 体态纠正/锻炼/阅读：每次投入时长
  - 交易复盘：思考时长
  - 早睡早起：睡眠时间 + 起床时间
- 自动累计：年度完成天数 + 总时长

### 3.2 每日成长日记
- 随时记录每日收获、心得、复盘反思、情绪变化
- 支持插入图片
- 支持标签管理
- 自动关联当天习惯数据
- 生成个人成长时间轴

### 3.3 年度热力图与统计看板
- 日历形式展示年度打卡状态
- 累计时长趋势图
- 连续坚持周数统计
- 月度对比分析

### 3.4 用户注册登录与云端备份
- 手机号 + 邮箱注册登录
- Firebase Authentication
- 数据自动同步至云端
- 换设备数据恢复

### 3.5 智能激励与成就系统
- 成长勋章：连续打卡天数突破、总时长里程碑、习惯均衡性奖励
- 里程碑庆祝动画
- 正向鼓励提醒通知

### 3.6 个性化目标推荐
- 基于历史表现推荐每日最低时长
- 动态调整习惯提醒时间
- 循序渐进巩固自律行为

## 4. UI/UX 设计方向

### 4.1 视觉风格
- **主题**：极简工业风 + 纯暗黑模式
- **设计语言**：Material Design 3
- **主色调**：金属银 (#C0C0C0) + 纯黑背景 (#000000)
- **强调色**：薄荷绿 (#98D4BB) 用于成功/打卡状态
- **背景色**：纯黑 (#000000) / 深灰黑 (#121212)

### 4.2 布局方式
- **底部导航**：首页、习惯、打卡、日记、我的
- **首页**：今日概览 + 快速打卡入口
- **习惯页**：五大习惯分类卡片
- **打卡页**：计时器 + 打卡记录
- **日记页**：时间轴式日记列表
- **我的**：个人中心 + 设置 + 数据导出

### 4.3 应用图标设计
- 纯黑背景 (#000000)
- 金属银色 (#C0C0C0) 连续线条构成几何字母 "L"
- 设计灵感：一条延伸向远方、然后急剧向上转折的长路
- 工业设计风格，纯几何线条，无霓虹发光效果

### 4.4 交互体验
- 卡片式打卡动效
- 成就解锁微动画
- 连续打卡火焰动画
- 热力图灰阶渐进变色
